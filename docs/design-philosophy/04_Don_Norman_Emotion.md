# 좋은 디자인이 행복을 주는 3가지 방식 - Don Norman

## 1. 감정의 3단계 처리 모델

1. **본능적 레벨 (Visceral Level)**: 생물학적으로 타고난 즉각적인 반응. 모양, 색상, 질감 등 외형에서 오는 즐거움. (예: 아름다운 물병, 재규어 자동차)
2. **행동적 레벨 (Behavioral Level)**: 사용의 즐거움. 효율성, 기능성, 그리고 사용자가 상황을 통제하고 있다는 느낌. (예: 잘 균형 잡힌 칼, 운전하기 즐거운 미니쿠퍼)
3. **반성적 레벨 (Reflective Level)**: 사용 후의 기억, 자아 이미지, 브랜드 가치. "이 제품이 나를 어떻게 정의하는가?"에 대한 답. (예: 환경 보호를 위해 비싼 전기차를 사는 행위)

## 2. 감정과 문제 해결

- **행복과 창의성**: 사람이 행복할 때(Positive Valence) 도파민이 분비되어 '확산적 사고(Breadth-first)'가 가능해진다. 즉, 더 창의적으로 문제를 해결할 수 있다.
- **불안과 집중**: 반대로 불안이나 두려움은 사고를 좁히고(Depth-first) 집중하게 만든다. 디자인은 사용자가 창의적이고 즐겁게 목표를 달성하도록 도와야 한다.

## 3. 결론: 기능 그 이상의 가치

과거에는 '사용성'만을 강조했으나, 이제 디자인은 '아름다움'과 '즐거움'을 포함해야 한다. 즐거운 물건은 실제로 더 잘 작동하는 것처럼 느껴지며, 사용자의 삶을 풍요롭게 만든다.
